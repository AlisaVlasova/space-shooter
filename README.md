# 🚀 Space Shooter (PIXI + TypeScript)

Гравець керує кораблем, уникає астероїдів і бореться з босом.

---

## 🔧 Команди

```bash
# встановити залежності
npm install

# запустити локально (http://localhost:5173)
npm run dev

# збілдити демо
npm run build

# деплой на GitHub Pages
npm run deploy
```

## 📦 Технології
	•	⚙️ Vite
	•	🎮 PIXI.js
	•	💬 TypeScript
	•	🧹 ESLint + Prettier
